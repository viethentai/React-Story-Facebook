# React-Story-Facebook-Extension
An extension that allows users to add other emojis instead of the default Facebook emojis to their Stories. Remake version from whoant source code: https://github.com/whoant/react-story-facebook
## Installation

Download and extract the extension. Navigate to **chrome://extensions/** and turn on developer mode. Click on 'unpacked extension' and then select the path containing the previously unpacked extension. Reload Facebook and go to the Story page.
